
## Features
  * API integration
  * Syncing Magento Users
  * Allowing Magento users to subscribe or unsubcribe

## Installation Notes
  * You need to have a Emma API Key ,Emma username and Emma password.

## Configuration
  1. Configure your Emma API details at Sytem/Configuration/Emma Setting in admin section to configure the module.

  2. You will need to put in your Emma API key Emma username and Emma password for your Emmaaccount.
  If you do not have a Emma account,You can generate it from https://app.e2ma.net/app2/billing/settings/
